# MARLM

## Components

This repo vendors two upstream projects in subdirectories, kept separate for clear attribution:

- [graphiti/](graphiti/) — from [getzep/graphiti](https://github.com/getzep/graphiti), Apache-2.0.
  A Python framework for temporally-aware knowledge graphs.
- [rlm/](rlm/) — from [alexzhang13/rlm](https://github.com/alexzhang13/rlm), MIT.

Upstream `LICENSE` files are preserved in each subdirectory. New work in this repo
is licensed Apache-2.0 — see [LICENSE](LICENSE) and [NOTICE](NOTICE).

## Modifications

Per Apache-2.0 §4(b), any modifications to files originating from `graphiti/`
will carry a prominent notice. Commit history in this repo is the canonical
record of changes from the upstream snapshot.
